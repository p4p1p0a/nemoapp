# Memo App Project Context (AI Memory)

## 概要
これは Windows 環境の Antigravity AI（私）から、Mac 環境等で新しく起動される同AIに対する「記憶の引き継ぎ（コンテキスト共有）」ドキュメントです。

## プロジェクトのミッション
**「NotionとObsidianとTldraw（無限キャンバス）の利点を融合した、ハイブリッド・メモアプリの構築」**

1. **データ構造・永続化**
   - デスクトップアプリ構築を見据え、ブラウザの localStorage 制限（5MB）を突破するため **Tauri v2** を用いたネイティブファイルシステム（`@tauri-apps/plugin-fs`）でのローカルファーストな保存機構を実装済み。
   - `storageAdapter.ts` でインターフェースを統合しているため、Webブラウザ起動時とTauri起動時で保存先が自動フォールバックする仕組み。
   - ノートツリーは親ノート・子ノートの概念を持つフラットJSON構造。日記はルートの`YYYY`フォルダから階層自動生成される仕様に変更済み。

2. **UI/UX 設計思想**
   - 黒を基調としたミニマルでプロフェッショナルなデザイン。
   - Tailwind CSS によるガラスモーフ（backdrop-blur）やなめらかな画面遷移（transition）を活用。
   - マークダウンエディタと、完全なフルスクリーン無限キャンバス（InfiniteBoard）の2つのモードを持つ。

3. **InfiniteBoard（無限キャンバス）の仕様**
   - ズーム（ホイール）とパン操作（手のひらツール・中ボタンドラッグ）による自由な視点移動。
   - ペンツール、消しゴムによるベクター描画機能。
   - 「👆 選択ツール (Select)」による、画像およびYouTube iframe の配置・移動・リサイズ機能。
   - **iframeイベントハック**: YouTube動画の半永久ドラッグバグを防ぐため、選択ツール時にはオーバーレイを挟み、**ダブルクリック（Interactive Mode）で初めて動画操作が可能になる**という、Figma/Miroライクな洗練された操作感を実装済み。
   - 変更履歴（HistoryStack）による Undo / Redo 機能と、ショートカットキー（Delete / Ctrl+Z）対応。

## Mac環境の私（AI）への引き継ぎ事項
このプロジェクトを開いたAntigravityは、まず `app/page.tsx` と `app/components/InfiniteBoard.tsx` のコードを読み込んでください。
